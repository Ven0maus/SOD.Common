\### Required IL2CPP DLLs

You can get these from your r2modman interop folder after running the game once.
Put these in this folder, so the project can compile properly.

* Assembly-CSharp.dll
* FMODUnity.dll
* Il2Cppmscorlib.dll
* Il2CppSystem.Core.dll
* Rewired\_Core.dll
* Unity.TextMeshPro.dll
* UnityEngine.CoreModule.dll
* UnityEngine.InputLegacyModule.dll
* UnityEngine.PhysicsModule.dll
* UnityEngine.UI.dll
* UnityEngine.dll

### Mod dlls required

* AssetBundleLoader.dll (from piepie's assetbundleloader mod)
* DialogUIRework.dll (from piepie's DialogUIRework mod)

(install the mod(s), then open the mod in the plugins folder, dll should be there)
